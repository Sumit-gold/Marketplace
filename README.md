# ServiGo - Local Service Marketplace

A premium, React-based local service marketplace platform.

## Tech Stack
- **Framework**: React + Vite
- **Styling**: Tailwind CSS + generic CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Routing**: React Router DOM

## Setup Instructions

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features
- **Modern UI/UX**: Glassmorphism, smooth animations, premium color palette.
- **Service Booking Flow**: Date/time selection, address, payment simulation.
- **Responsive Design**: Mobile-first approach.
- **Dashboard**: Manage bookings and profile.
